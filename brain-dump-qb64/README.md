# 🧠 Brain Dump QB64

**Brain Dump** is a retro-style idea organizer built using **QB64 Phoenix Edition**. It lets you quickly record and manage your thoughts, ideas, and mental notes with a nostalgic text-based interface.

## 🚀 Features

- Write and save ideas to disk
- View and delete existing entries
- Retro-style terminal UI
- Works offline
- Pure QB64 — no external tools required

## 📂 Project Structure

```
brain-dump-qb64/
├── main.bas            # Main entry point and menu
├── menu_utils.bas      # Menu display and navigation helpers
├── idea_manager.bas    # Logic for managing ideas
├── file_manager.bas    # File saving/loading functions
└── ideas/              # (Created at runtime) Folder for saved entries
```

## 🛠 Requirements

- [QB64 Phoenix Edition](https://www.qb64phoenix.com/)

## ▶️ How to Run

1. Open `main.bas` in QB64 Phoenix Edition.
2. Press F5 to compile and run.
3. Use the on-screen menu to start adding ideas.
